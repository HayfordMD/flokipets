# Generator UI in Marketplace

**Status:** TODO

**Description:**
Create a standalone section within the Marketplace for the Generator. It should display the total pool of donated gas, recent donations, and allow the user to deposit gas/Floki into the Burner to fund the community. 

**Acceptance Criteria:**
- The page is standalone but linked/accessible from the Marketplace.
- Displays current gas levels in the Generator contract (read via Thirdweb SDK).
- Contains a UI flow for players to donate Gas (BNB) or Floki.
- UI button/flow for "Convert to On-Chain Floki" which triggers the backend conversion request.
